# Membership

The `Membership` type represents the membership level associated with a game or user account.

| Label     | Value       | Description                                           |
| :-------- | :---------- | :---------------------------------------------------- |
| `NONE`    | `"NONE"`    | No membership subscription associated with the game.  |
| `PS_PLUS` | `"PS_PLUS"` | PlayStation Plus membership associated with the game. |
