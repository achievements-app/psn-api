# TitlePlatform

The `TitlePlatform` type represents the gaming platform a title is available on.

| Label  | Value    | Description                         |
| :----- | :------- | :---------------------------------- |
| `PS5`  | `"PS5"`  | PlayStation 5 platform.             |
| `PS4`  | `"PS4"`  | PlayStation 4 platform.             |
| `PS3`  | `"PS3"`  | PlayStation 3 platform.             |
| `Vita` | `"Vita"` | PlayStation Vita handheld platform. |
